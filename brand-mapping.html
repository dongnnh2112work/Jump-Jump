ASSETS
Brand Asset Checklist — cần cung cấp
Logo
  • SVG hoặc PNG nền trong
  • Tối thiểu 200×200px
  • Full logo + icon-only (vuông)
  • Trên nền tối và nền sáng
Brand Name
  • Tên chính thức (plain text)
  • Tên rút gọn nếu có
  • Tên game custom nếu muốn
Primary Color
  • Mã HEX chính xác
  • Dùng: block top, button border, selection ring, sky gradient
  • VD: Nike #111111, Coca-Cola #CC0000
Secondary Color
  • HEX phụ (thường tối hơn primary)
  • Dùng: block side faces, gradient sky 2
Font
  • Display font (tiêu đề, score)
  • Body font (UI, hướng dẫn)
  • Google Fonts link hoặc .woff2
  • Optional — giữ Fredoka One + Nunito
Tagline / Copy
  • Tagline ngắn ≤40 ký tự
  • CTA custom nếu có
  • Tên event / campaign
Character Skin
  • Chọn preset hoặc yêu cầu custom
  • Custom: màu áo, quần, giày
  • Logo ngực áo ≤3 ký tự hoặc SVG icon
Background / Theme
  • Chọn: Cloud Garden / Arctic Peak / Lantern Dusk
  • Hoặc custom 2 hex màu sky
  • Optional: ảnh nền attract screen
